
- Install Debian
- Install Docker
- Open Nextcloud github compose.yaml:
	- Comment out port 80:80/8443:8443
	- Uncomment APACHE_PORT and APACHE_IP_BINDING
	- SKIP_DOMAIN_VALIDATION: true
	- Uncomment caddy/configs
